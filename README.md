# ssshell
linux shell

## Compile and Run
**compile**:	gcc shell.c -lreadline -o shell<br>
**run**:		./shell

## Libraries Used
* <stdio.h> 
* <string.h> 
* <stdlib.h> 
* <unistd.h> 
* <sys/types.h> 
* <sys/wait.h> 
* <readline/readline.h> 
* <readline/history.h><br><br>
_for retrieving hostname and ip address_
* <errno.h> 
* <netdb.h> 
* <sys/types.h> 
* <sys/socket.h> 
* <netinet/in.h> 
* <arpa/inet.h>
